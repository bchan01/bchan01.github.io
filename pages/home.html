Exposing Legacy SOAP Web Services as REST API using Node

A simple framework developed in Node.JS to wrap SOAP Web service as RESTful Web Service without dealing with WDSL and schemas. All you need to do is to generate a SOAP envelope from tools like SOAP UI and use it as a template to make SOAP call via HTTP POST. This framework will take care of XML to JSON and JSON to XML transformations.
