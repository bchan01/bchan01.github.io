Data Masking using Python
This project illustrates how easy it is to mask sensitive data from a real production dataset to comply with user privacy law. This tool can also be used to generate massive amount of data for testing purpose.
Data Exploration with Spark and Zeppelin
I develop a Notebook to show how easy it is to retrieve stock prices to perform analysis and visualization. It was built and tested with Zeppelin provided by Hortonworks Sandbox.
Exposing Legacy SOAP Web Services as REST API using Node
A simple framework developed in Node.JS to wrap SOAP Web service as RESTful Web Service without dealing with WDSL and schemas. All you need to do is to generate a SOAP envelope from tools like SOAP UI and use it as a template to make SOAP call via HTTP POST. This framework will take care of XML to JSON and JSON to XML transformations.
