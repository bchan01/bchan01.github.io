Data Masking using Python

This project illustrates how easy it is to mask sensitive data from a real production dataset to comply with user privacy law. It also include utilities for generating test data

Exposing Legacy SOAP Web Services as REST API using Node

A simple framework developed in Node to wrap SOAP Web services as RESTful services without dealing with WDSL and schemas. All that is required is a generated SOAP envelope from tools like SOAP UI. The SOAP envelope is used as a template to make SOAP call via HTTP POST. This framework takes care of the trivial XML to JSON and JSON to XML transformations.

Stock Portfolio Tracker Application (MEAN Stack)

I am developing a set of Microservices (RESTful API) to allow user to create stock portfolio and track its performance over time. This project is still under development. I am hoping to develop a user interface in the near future.
